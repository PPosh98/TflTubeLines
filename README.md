## About
This is an Android app that simply displays the name and statuses of all the London tube lines.

## Technologies
- MVVM Architecture
- Dagger & Hilt for Dependency Injection
- Retrofit + OkHttp for API handling
- Kotlin
- Coroutines, Live Data
- XML for UI
